# EX-07 Development of a Prompt-Based Application using Large Language Models

**Opened:** 17 May 2026, 10:00 AM  
**Due:** 31 May 2026, 11:59 PM

---

# Aim

To demonstrate how to create a prompt-based application to organize daily tasks, showing the progression from simple prompts to more advanced prompt designs and their corresponding outputs.

---

# Objective

To develop a prompt-based application tailored to personal needs using Large Language Models (LLMs), fostering creativity and practical problem-solving skills.

Learners should understand:
- Prompt design techniques
- Application-oriented prompting
- Refinement of prompts
- AI-generated outputs and analysis

---

# Experiment

Develop a prompt-based application using ChatGPT or similar AI tools.

Learners can choose application ideas such as:
- Study Time Scheduler
- Project Task Breakdown
- Personal Budget Manager
- Travel Itinerary Planner
- Daily Productivity Assistant
- Fitness or Meal Planner

---

# Algorithm

## Step 1: Identify the Problem
- Select a daily-life problem or task.
- Define the objective of the application.

## Step 2: Design Basic Prompt
- Create a simple prompt for the selected task.
- Generate initial output.

## Step 3: Refine the Prompt
- Add constraints, formatting, and detailed instructions.
- Improve prompt clarity.

## Step 4: Generate Advanced Outputs
- Use advanced prompts for better personalization.
- Compare outputs from simple and refined prompts.

## Step 5: Analyze the Results
- Evaluate effectiveness of prompts.
- Identify improvements in output quality.

## Step 6: Document the Application
- Include prompts
- Generated outputs
- Screenshots
- Observations

Learners should generate:
- Prompt-based application outputs
- Simple and refined prompt comparisons
- Personalized AI-generated responses
- Screenshots and observations
