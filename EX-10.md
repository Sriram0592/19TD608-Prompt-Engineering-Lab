# EX-10 Content Creation Using Prompt Patterns

**Opened:** 11 March 2026, 08:00 AM  
**Due:** 17 March 2026, 11:00 PM

---

# Aim

To demonstrate how various prompting techniques can be employed to create different forms of content such as reports, articles, case studies, and creative works using AI tools.

---

# Objective

To highlight how different prompt structures affect the content’s quality, coherence, creativity, and structure.

Learners will explore prompt engineering techniques such as:
- Query decomposition
- Decision-making prompts
- Semantic filtering
- Structured prompting
- Creative prompting

---

# Experiment

Learners must use AI tools such as ChatGPT or similar models to generate various forms of content including:
- Reports
- Articles
- Case Studies
- Blog Posts
- Creative Stories
- Comic or Script Ideas

The experiment should demonstrate how prompt design impacts:
- Content clarity
- Creativity
- Structure
- Quality of information
- Coherence

---

# Algorithm

## Step 1: Select Content Type
- Choose content category such as report, article, or case study.

## Step 2: Create Initial Prompt
- Write a basic prompt for content generation.

## Step 3: Generate Initial Output
- Use AI tool to create content.

## Step 4: Refine the Prompt
- Improve prompt using structured instructions.
- Add constraints and formatting.

## Step 5: Generate Improved Content
- Produce refined outputs using advanced prompt patterns.

## Step 6: Compare Outputs
- Analyze differences between simple and refined prompts.

## Step 7: Document Observations
- Evaluate content quality and effectiveness.
